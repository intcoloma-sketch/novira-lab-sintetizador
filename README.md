# Sintetizador Web — Prueba 1

Abrir `index.html` en un navegador moderno. Usa el teclado (z, s, x, d, c, v, g, b, h, n, j, m, , , l, . , ; , /) o haz click/touch en las teclas para tocar.

Recomendado: servir con un servidor local para evitar restricciones de CORS al cargar desde filesystem.

Ejemplo rápido:

```bash
# desde la carpeta "Prueba 1"
python3 -m http.server 8000
# luego abrir http://localhost:8000
```

Controles añadidos:
- Presets: `80s`, `Ambient`, `Piano eléctrico`
- Delay y Reverb con controles en tiempo real
- Teclado clickable/touch + soporte de teclado físico
